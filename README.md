# Kali-Linux-Phishing-Simulation

A demonstration environment and toolkit for simulating phishing attacks to understand social engineering tactics, credential harvesting, and how to implement effective organizational defenses.

Disclaimer: This project is for educational and authorized security testing purposes only. Usage of these tools for attacking targets without prior mutual consent is illegal. The author assumes no liability for any misuse or damage caused by this program.

This project utilizes Kali Linux and various open-source social engineering tools to simulate real-world phishing scenarios. By recreating these attacks in a controlled environment, users can learn to identify malicious URLs, suspicious login pages, and the mechanics of credential theft.
